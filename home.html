SPARK UI

Create accessible React apps with speed

Spark UI is a simple, modular and accessible component library that gives you the building blocks you need to build your React applications.

Installation

Copy and paste the code in the head tag of your html document. So that you can style your components by just adding class names to your html elements. 


          <link rel="stylesheet"
          href="https://uispark.netlify.app/allComponents.css" >
